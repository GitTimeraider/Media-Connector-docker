# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-28

### Added
- Initial release of Media Connector
- Full Sonarr integration (TV shows management)
- Full Radarr integration (Movies management)
- Lidarr integration (Music management)
- Readarr integration (Books management)
- SABnzbd download client support
- NZBGet download client support
- qBittorrent download client support
- Transmission download client support
- Deluge download client support
- Overseerr integration (Media requests)
- Tautulli integration (Plex monitoring)
- Prowlarr integration (Indexer management)
- Jackett integration (Torrent proxy)
- Responsive Material-UI web interface
- Dark theme
- Dashboard with statistics
- Downloads page with real-time progress
- Service overview page
- Settings page with service configuration
- Docker support with automated builds
- Multi-architecture Docker images (amd64, arm64)
- GitHub Actions CI/CD pipeline
- Automatic Docker image publishing to ghcr.io
- Comprehensive documentation

### Features
- Add, edit, and delete service instances
- Test service connections
- Browse TV shows and movies library
- Search for new content
- Monitor downloads in real-time
- View queue status
- Calendar view for upcoming releases
- Service health monitoring
- Secure API key authentication
